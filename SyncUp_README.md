# 🚀 What is SyncUp?

**SyncUp** is a prototype social system that matches gamers based on
their **Game Personality**.

Instead of random teammates, the system builds a **player profile** that
describes:

-   your play rhythm\
-   your cooperation style\
-   your game preferences\
-   your activity level

Then it matches you with someone whose gaming personality **fits
yours**.

Think of it as:

    MBTI for gamers + matchmaking + voice room

# 🧠 Game Personality System

At the core of SyncUp is a lightweight personality model for gamers.

Users answer **16 quick questions** to generate a **Game Personality
Code**.

Example:

    BTR‑F

### Structure

    Two core traits + one activity tag

  Dimension    Meaning
  ------------ --------------------------------------------
  Playstyle    A Aggressive / B Balanced / S Supportive
  Preference   C Competitive / T Teamplay / E Exploration
  Activity     F Frequent Player / L Casual Player

### Example Interpretation

    BTR‑F
    Balanced · Teamplay · Growth oriented · Frequent player

This player:

-   values strategy and timing
-   prefers coordinated team play
-   enjoys long‑term progression
-   plays frequently

This allows SyncUp to match players with **compatible expectations**.

## 🎮 Smart Teammate Matching

Instead of random teammates, you define:

-   game title
-   playstyle
-   preferred teammate type
-   team size

Example:

    Game: League of Legends
    Mode: Duo Queue
    Preference: High Communication

SyncUp finds players with **similar game personalities**.

------------------------------------------------------------------------

## 🗣 Voice Lobby

After matching:

-   a voice room is created
-   teammates can chat first
-   confirm roles & strategy
-   then start the game

This recreates the feeling of:

    friends grouping up
    instead of
    random matchmaking
