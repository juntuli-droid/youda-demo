# SyncUp 游戏主题 UI 设计规范

## 设计方向
- 视觉基调：深色界面 + 霓虹蓝紫强调色
- 风格锚点：语音协作工具的低干扰信息密度 + Riot 风格的竞技感层次
- 体验目标：高对比可读、低学习成本、关键动作高聚焦

## 主题令牌
- 背景色：`--bg-0`, `--bg-1`, `--bg-2`
- 面板色：`--panel`, `--panel-soft`
- 线框色：`--line`
- 主文案：`--text-main`
- 次文案：`--text-sub`
- 品牌渐变：`--brand-1`, `--brand-2`
- 功能色：`--accent`, `--success`, `--danger`

## 核心组件风格
- 容器：`game-shell`, `game-sidebar`, `game-panel`
- 主按钮：`neon-btn`
- 次按钮：`neon-outline-btn`
- 输入框：`neon-input`
- 标签：`hud-label`, `neon-chip`
- 动效：`motion-fade-in`

## 交互规范
- 主按钮 hover 需抬升 1px 并增强发光
- 输入框 focus 需提供可见边框与外环
- 错误信息统一为红色描边 + 半透明底
- 状态标签统一采用圆角胶囊样式

## 响应式策略
- 侧栏固定宽度 80px
- 主内容在小屏使用单列堆叠
- 关键操作按钮保持满宽，提升触控可达性

## 页面映射
- 首页、登录、注册、人格结果、个人主页已采用主题令牌
- 匹配与房间页沿用同一主题变量，可继续平滑扩展
