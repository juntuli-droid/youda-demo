# 全站视觉一致性清单

## 已统一页面
- 首页 `/`
- 注册 `/register`
- 登录 `/login`
- 人格入口 `/personality`
- 人格问卷 `/personality/questions`
- 人格结果 `/personality/result`
- 个人主页 `/profile`
- 匹配配置 `/match`
- 匹配中 `/matching`
- 房间 `/room`
- 生涯 `/career`
- Vlog `/vlog`

## 统一规则
- 背景：游戏素材 + 深色渐变叠加
- 侧栏：`game-sidebar`
- 主容器：`game-panel`
- 主按钮：`neon-btn`
- 次按钮：`neon-outline-btn`
- 输入控件：`neon-input`
- 标签：`neon-chip`

## 关键视觉参数
- 主色：`#0E1A25`
- 辅色：`#10B981`
- 卡片圆角：`8px`
- 卡片阴影：`0 4 12 0 rgba(0,0,0,0.15)`

## 验证建议
- 分辨率：1920×1080 / 1366×768 / 375×667
- 检查项：
  - 标题与正文无截断
  - 按钮 hover 和禁用态一致
  - 卡片层级和边框透明度一致
  - 头像、角色图无拉伸变形
