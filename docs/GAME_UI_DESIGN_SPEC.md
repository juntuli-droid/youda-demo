# 游戏结果页设计规范（Kook / Riot 风格提炼）

## 栅格与间距
- 基础单位：8px
- 间距节奏：8 / 16 / 24 / 32
- 容器最大宽：1200px（`max-w-6xl`）
- 主区块结构：Hero / Metrics / Bottom Actions 三段

## 字体层级
- H1：30 / 26 / 22
- 标题副文案：16 / 15 / 14
- 正文：14（行高 1.7）
- 标签：12-13

## 视觉风格
- 卡片式布局，圆角 8
- 玻璃化深色背景 + 柔和边框
- 品牌主按钮渐变：绿-青
- 次按钮低对比描边强调

## 动效曲线
- 进入：`ease` 360ms
- Hover：`transform` 160-180ms

## 无障碍与对比
- 文字与背景最小对比接近 WCAG AA
- 操作按钮高度统一 48px
- 触控目标充足，不低于 40px

## 响应式断点
- Mobile `<640`
- Tablet `640-1023`
- Desktop `>=1024`
- Wide `>=1280`

## RTL 与主题
- 支持 `dir=rtl`：文本采用 `text-start` 语义对齐
- 主题 token 支持深色 / 浅色切换，默认深色
